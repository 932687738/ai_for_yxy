## 0.1 概念讲义：AI 数学术语与核心知识详解

### 0.1.1 第 2 课到底在学什么

第 2 课的任务不是让你成为数学专家，而是让你看懂后续机器学习和深度学习中反复出现的数学语言。

机器学习中经常出现：

```text
X
y
w
b
loss
gradient
learning_rate
embedding
similarity
```

这些符号背后对应的就是：

- 向量。
- 矩阵。
- 概率。
- 统计。
- 导数。
- 梯度。
- 损失函数。

【重点】本课的目标是建立“数学概念和 AI 场景之间的映射关系”。

### 0.1.2 标量 Scalar

标量就是一个单独的数。

例如：

```text
价格 = 120
概率 = 0.82
学习率 = 0.01
损失值 = 3.5
```

在 AI 中，标量常用于表示：

- 一个特征值。
- 一个概率。
- 一个损失值。
- 一个参数。
- 一个超参数。

【重点】标量是最小的数值单位。

### 0.1.3 向量 Vector

向量是一组有顺序的数。

一个房子可以表示为：

```text
[面积, 房间数, 房龄]
```

一个用户可以表示为：

```text
[最近登录天数, 订单数, 累计消费金额]
```

一段文本也可以通过 Embedding 表示为：

```text
[0.12, -0.33, 0.85, ...]
```

【核心】向量是 AI 中表示一个对象的常见方式。

你可以把向量理解为：

```text
把一个现实对象拆成多个数字维度后的表示。
```

### 0.1.4 维度 Dimension

维度表示向量里有多少个数字。

例如：

```text
[100, 3, 8]
```

是 3 维向量。

Embedding 可能是：

```text
1536 维向量
3072 维向量
```

【重点】维度越高，能表达的信息可能越丰富，但计算和理解也更复杂。

【易错】维度高不等于一定更好，关键看表示是否有效。

### 0.1.5 矩阵 Matrix

矩阵是二维数字表格。

在机器学习中，特征矩阵通常长这样：

```text
每一行：一个样本
每一列：一个特征
```

例如：

```text
[
  [80, 2, 10],
  [100, 3, 8],
  [120, 3, 5]
]
```

表示 3 个房子，每个房子 3 个特征。

【核心】矩阵是批量样本的数字表示。

### 0.1.6 张量 Tensor

张量可以理解为多维数组。

```text
0 维：标量
1 维：向量
2 维：矩阵
3 维及以上：张量
```

图片数据常用张量表示：

```text
高度 x 宽度 x 颜色通道
```

深度学习框架 PyTorch 中最核心的数据对象就叫 Tensor。

【重点】深度学习中的数据、参数、中间计算结果，通常都以张量形式存在。

### 0.1.7 shape

shape 表示数组、矩阵、张量的形状。

例如：

```text
X.shape = (100, 5)
```

通常表示：

```text
100 个样本
5 个特征
```

【核心】shape 是调试 AI 代码最重要的信息之一。

【易错】很多 AI 代码报错不是算法错，而是 shape 不匹配。

### 0.1.8 点积 Dot Product

点积是两个向量对应元素相乘后求和。

例如：

```text
[1, 2, 3] · [10, 20, 30]
= 1*10 + 2*20 + 3*30
= 140
```

在 AI 中，点积常用于：

- 线性模型预测。
- 神经网络加权求和。
- 计算向量相关性。
- Attention 机制中的相似度计算。

【核心】点积是“特征值和权重共同作用”的数学表达。

### 0.1.9 权重 Weight

权重表示某个特征对预测结果的影响强度。

例如：

```text
price = 1.5 * area + 10 * rooms - 2 * age
```

其中：

```text
1.5、10、-2
```

都是权重。

【重点】模型训练的一个重要目标，就是学习合适的权重。

### 0.1.10 偏置 Bias

偏置是模型中的常数项。

例如：

```text
price = 1.5 * area + 10 * rooms - 2 * age + 5
```

这里的 `5` 就是偏置。

它可以理解为模型的基础起点。

【重点】线性模型通常由权重和偏置共同决定预测结果。

### 0.1.11 范数 Norm

范数可以理解为向量的长度。

例如二维向量：

```text
[3, 4]
```

长度是：

```text
5
```

因为：

```text
sqrt(3^2 + 4^2) = 5
```

在 AI 中，范数常用于：

- 计算距离。
- 归一化向量。
- 正则化模型参数。
- 计算余弦相似度。

### 0.1.12 欧氏距离 Euclidean Distance

欧氏距离衡量两个点之间的直线距离。

它回答：

```text
两个向量在空间中离得有多远？
```

在推荐、聚类、检索中，距离常用于衡量样本相似程度。

【重点】距离小通常表示更接近，但在高维语义向量中，余弦相似度更常见。

### 0.1.13 余弦相似度 Cosine Similarity

余弦相似度衡量两个向量方向是否接近。

它不重点关注长度，而关注方向。

在 Embedding 检索中：

```text
问题向量和文档向量方向越接近，语义越相似。
```

【核心】RAG 和语义搜索的基础之一就是向量相似度。

【易错】余弦相似度不是判断文本字面是否相同，而是判断向量表示的语义是否接近。

### 0.1.14 Embedding

Embedding 是把文本、图片、商品、用户等对象转换成向量表示。

例如：

```text
"Java AI 应用开发" -> [0.12, -0.08, 0.91, ...]
```

Embedding 的价值是：

```text
把非结构化对象变成可以计算相似度的数字向量。
```

【核心】Embedding 是大模型应用、语义搜索、RAG 的基础概念。

### 0.1.15 概率 Probability

概率描述不确定性。

例如：

```text
用户流失概率 = 0.82
邮件是垃圾邮件概率 = 0.97
图片是猫的概率 = 0.91
```

概率不是绝对事实，而是模型在现有信息下的判断。

【重点】模型输出概率后，业务系统通常还需要根据阈值做决策。

### 0.1.16 条件概率 Conditional Probability

条件概率表示在某个条件发生的情况下，另一个事件发生的概率。

例如：

```text
P(用户流失 | 最近 30 天未登录)
```

意思是：

```text
已知用户最近 30 天未登录，该用户流失的概率。
```

【重点】很多机器学习预测可以理解为在给定特征条件下估计目标发生的概率。

### 0.1.17 随机变量 Random Variable

随机变量是结果不确定的变量。

例如：

```text
明天某商品销量
某用户是否流失
一次点击是否转化
```

这些值在发生前是不确定的，只能用概率描述。

### 0.1.18 均值 Mean

均值表示一组数据的平均水平。

例如：

```text
平均房价
平均订单金额
平均登录次数
```

【重点】均值能描述整体水平，但容易受极端值影响。

### 0.1.19 方差 Variance 和标准差 Standard Deviation

方差和标准差描述数据波动程度。

如果标准差大，说明数据分散。

如果标准差小，说明数据集中。

在 AI 中，它们用于：

- 判断数据是否稳定。
- 发现异常值。
- 理解特征分布。
- 做标准化处理。

### 0.1.20 分布 Distribution

分布描述数据出现的整体形态。

例如：

```text
房价集中在哪个区间？
大部分用户消费金额是多少？
是否少数用户消费特别高？
类别是否严重不平衡？
```

【核心】训练模型前必须先观察数据分布。

### 0.1.21 导数 Derivative

导数描述变化率。

它回答：

```text
当 x 变化一点点时，y 会变化多少？
```

在模型训练中，我们关心：

```text
参数变化一点点，loss 会变大还是变小？
```

【核心】导数告诉我们参数调整的方向信息。

### 0.1.22 梯度 Gradient

梯度是多个参数对应导数组成的向量。

如果模型有多个参数：

```text
w1, w2, w3, b
```

那么梯度可能是：

```text
[dw1, dw2, dw3, db]
```

【核心】梯度告诉模型每个参数应该如何调整，才能让损失下降。

### 0.1.23 损失函数 Loss Function

损失函数衡量模型预测有多错。

例如：

```text
真实房价 = 150
预测房价 = 130
误差 = -20
```

损失函数会把误差转换成一个可优化的数值。

【核心】模型训练的目标就是让损失函数尽可能小。

### 0.1.24 MSE

MSE 是均方误差。

它先把误差平方，再求平均。

适合回归任务。

【重点】平方会放大大误差，所以 MSE 对大错误更敏感。

### 0.1.25 学习率 Learning Rate

学习率控制每次参数更新的步长。

学习率太小：

```text
训练很慢。
```

学习率太大：

```text
可能震荡或发散。
```

【核心】学习率是训练模型时最重要的超参数之一。

### 0.1.26 梯度下降 Gradient Descent

梯度下降是一种优化方法。

它的目标是：

```text
沿着让 loss 下降的方向，一步步更新参数。
```

参数更新公式可以理解为：

```text
新参数 = 旧参数 - 学习率 * 梯度
```

【核心】梯度下降是理解机器学习训练过程的基础。

【核心】Embedding 把文本等对象转换成可计算相似度的向量。

【核心】概率用于表达不确定性。

【核心】梯度告诉模型参数应该往哪个方向调整。

【核心】梯度下降通过不断更新参数来降低损失。

【重点】均值、方差、标准差帮助你理解数据整体水平和波动。

## 4. 标量、向量、矩阵、张量

### 4.1 标量 Scalar

标量就是一个单独的数。

例如：

```text
年龄 = 30
价格 = 99.5
学习率 = 0.01
损失值 = 2.35
```

Python 示例：

```python
age = 30
learning_rate = 0.01
loss = 2.35
```

在 AI 中，标量常用于：

- 一个特征值
- 一个模型参数
- 一个损失值
- 一个概率值
- 一个学习率

### 4.2 向量 Vector

向量是一组有顺序的数。

例如，一个房子可以用 3 个特征表示：

```text
面积 = 100
房间数 = 3
房龄 = 8
```

可以写成向量：

```text
[100, 3, 8]
```

Python / NumPy 表示：

```python
import numpy as np

house = np.array([100, 3, 8])
print(house)
print(house.shape)
```

输出：

```text
[100   3   8]
(3,)
```

在机器学习中，一个样本通常就是一个向量。

```text
一个用户 = [年龄, 消费次数, 最近登录天数, 是否会员]
一个商品 = [价格, 类目编号, 销量, 评分]
一段文本 = [0.12, -0.33, 0.85, ...]  # Embedding 向量
```

### 4.3 矩阵 Matrix

矩阵是二维数字表格。

如果有 5 套房子，每套房子 3 个特征：

```text
[80, 2, 10]
[100, 3, 8]
[120, 3, 5]
[150, 4, 3]
[60, 1, 15]
```

可以组成矩阵：

```python
import numpy as np

houses = np.array([
    [80, 2, 10],
    [100, 3, 8],
    [120, 3, 5],
    [150, 4, 3],
    [60, 1, 15],
])

print(houses.shape)
```

输出：

```text
(5, 3)
```

含义：

```text
5 行：5 个样本
3 列：3 个特征
```

机器学习中常用记号：

```text
X = 输入特征矩阵
y = 目标值
```

例如：

```text
X.shape = (样本数量, 特征数量)
y.shape = (样本数量,)
```

### 4.4 张量 Tensor

张量可以简单理解为多维数组。

```text
0 维张量：标量
1 维张量：向量
2 维张量：矩阵
3 维及以上：更高维数组
```

图片可以看成张量：

```text
灰度图片：高度 x 宽度
彩色图片：高度 x 宽度 x 颜色通道
一批彩色图片：批次大小 x 高度 x 宽度 x 颜色通道
```

深度学习框架 PyTorch 里最核心的数据结构就叫 Tensor。

## 5. shape：AI 代码里最重要的调试信息

AI 初学者写代码最常见的错误之一是维度不匹配。

例如矩阵乘法要求：

```text
(m, n) @ (n, p) = (m, p)
```

中间的 `n` 必须相同。

示例：

```python
import numpy as np

X = np.array([
    [80, 2, 10],
    [100, 3, 8],
])

w = np.array([1.5, 10, -2])

print(X.shape)
print(w.shape)

y_pred = X @ w
print(y_pred)
print(y_pred.shape)
```

输出：

```text
(2, 3)
(3,)
[120. 164.]
(2,)
```

解释：

```text
X 有 2 个样本，每个样本 3 个特征
w 有 3 个权重
X @ w 会得到每个样本的预测值
```

调试 AI 代码时，建议频繁打印：

```python
print(X.shape)
print(y.shape)
print(weights.shape)
```

## 6. 向量运算

### 6.1 向量加法


### 6.3 点积 Dot Product

点积是两个向量对应元素相乘后求和。

```python
import numpy as np

a = np.array([1, 2, 3])
b = np.array([10, 20, 30])

dot = np.dot(a, b)
print(dot)
```

计算过程：

```text
1*10 + 2*20 + 3*30 = 140
```

点积在 AI 中非常重要。

典型场景：

1. 线性模型预测。
2. 神经网络加权求和。
3. 向量相似度计算。
4. Attention 机制中的相关性计算。

线性模型可以写成：

```text
y = x1*w1 + x2*w2 + x3*w3 + b
```

用向量写就是：

```text
y = x · w + b
```

Python 示例：

```python
import numpy as np

print(y_pred)
```

计算过程：

```text
100*1.5 + 3*10 + 8*(-2) + 5 = 169
```

这就是一个极简房价预测公式。

## 7. 向量长度、距离与相似度

### 7.1 向量长度 Norm

二维向量 `[3, 4]` 的长度是 5，因为：

```text
sqrt(3^2 + 4^2) = 5
```

NumPy 示例：

```python
import numpy as np

v = np.array([3, 4])
length = np.linalg.norm(v)
print(length)
```

输出：

```text
5.0
```

更一般地，向量 `[x1, x2, ..., xn]` 的长度是：

```text
sqrt(x1^2 + x2^2 + ... + xn^2)
```

### 7.2 欧氏距离

欧氏距离衡量两个点之间的直线距离。

```python
import numpy as np

a = np.array([1, 2])
b = np.array([4, 6])

distance = np.linalg.norm(a - b)
print(distance)
```

计算：

```text
a - b = [-3, -4]
sqrt((-3)^2 + (-4)^2) = 5
```

### 7.3 余弦相似度

余弦相似度衡量两个向量方向是否接近。

公式：

```text
cosine_similarity(a, b) = (a · b) / (||a|| * ||b||)
```

Python 实现：

```python
import numpy as np


def cosine_similarity(a: np.ndarray, b: np.ndarray) -> float:
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))


query = np.array([1.0, 0.9, 0.1])
doc_a = np.array([0.9, 0.8, 0.2])
doc_b = np.array([0.1, 0.2, 0.9])

print(cosine_similarity(query, doc_a))
print(cosine_similarity(query, doc_b))
```

`doc_a` 和 `query` 更相似，因为它们方向更接近。

这就是向量检索和 RAG 的基础思想：

```text
用户问题 -> Embedding 向量
文档片段 -> Embedding 向量
计算问题向量和文档向量的相似度
返回最相似的文档片段
```

## 10. 概率基础

### 10.1 概率是什么

概率用于描述不确定性。

例如：

```text
明天下雨的概率是 70%
用户流失的概率是 20%
这张图片是猫的概率是 95%
这封邮件是垃圾邮件的概率是 98%
```

概率值范围：

```text
0 <= P <= 1
```

含义：

```text
0：不可能发生
1：一定发生
0.5：一半可能
```

### 10.2 分类模型输出为什么是概率

一个二分类模型可能输出：

```text
用户流失概率 = 0.82
```

业务系统可以据此决策：

```text
if probability >= 0.8:
    标记为高流失风险用户
else:
    正常跟进
```

多分类模型可能输出：

```text
猫：0.80
狗：0.15
兔子：0.05
```

概率最高的类别通常作为预测结果。

### 10.3 条件概率

条件概率表示“在某个条件已经发生的情况下，另一个事件发生的概率”。

记作：

```text
P(A | B)
```

读作：

```text
在 B 发生的条件下，A 发生的概率
```

业务例子：

```text
P(用户流失 | 最近 30 天未登录)
```

这表示：

```text
已知用户最近 30 天未登录的情况下，该用户流失的概率
```

机器学习里很多判断都可以理解为条件概率估计。

### 10.4 期望、方差、标准差

假设一组分数：

```python
import numpy as np

scores = np.array([60, 70, 80, 90, 100])

print(np.mean(scores))
print(np.var(scores))
print(np.std(scores))
```

含义：

| 指标 | 含义 |
|---|---|
| 均值 mean | 平均水平 |
| 方差 variance | 数据波动程度 |
| 标准差 standard deviation | 数据离均值的典型距离 |

如果两个班平均分都是 80：

```text
A 班：78, 79, 80, 81, 82
B 班：50, 70, 80, 90, 110
```

平均分相同，但 B 班波动更大，标准差更大。

AI 数据分析中，标准差常用于判断：

- 数据是否分散
- 是否存在异常值
- 特征是否需要标准化

## 11. 数据分布

### 11.1 为什么要看分布

训练模型前，不要急着写训练代码。先看数据分布。

例如房价数据：

```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("data/houses.csv")

plt.hist(df["price"], bins=5)
plt.xlabel("Price")
plt.ylabel("Count")
plt.title("Price Distribution")
plt.show()
```

你需要观察：

1. 数据是否集中在某个范围。
2. 是否有极端异常值。
3. 是否严重偏斜。
4. 不同类别样本是否均衡。

### 11.2 正态分布直觉

正态分布常被称为钟形曲线。

```text
          *
        *   *
      *       *
    *           *
  *               *
*                   *
```

很多自然现象近似正态分布，例如身高、测量误差等。

但真实业务数据不一定正态。例如：

- 用户消费金额可能长尾分布。
- 商品销量可能少数商品很高，大量商品很低。
- 日志访问量可能有明显峰值。

模型训练前了解数据分布，可以帮助你选择合适的处理方式。

## 12. 导数与变化率

### 12.1 导数的直觉

导数描述的是“变化率”。

如果函数是：

```text
y = x^2
```

当 x 变化一点点时，y 会变化多少？

这就是导数关心的问题。

例如：

```text
y = x^2
y' = 2x
```

当：

```text
x = 3
```

导数：

```text
y' = 2 * 3 = 6
```

含义：

```text
在 x = 3 附近，x 每增加一点，y 大约以 6 倍速度增加
```

### 12.2 为什么 AI 需要导数

模型训练要解决的问题是：

```text
怎样调整参数，才能让预测误差更小？
```

如果损失函数是：

```text
loss = (prediction - target)^2
```

我们需要知道：

```text
参数 w 增大一点，loss 会变大还是变小？
参数 b 增大一点，loss 会变大还是变小？
```

导数告诉我们变化方向和变化速度。

### 12.3 梯度

如果只有一个参数，导数是一个数。

如果有多个参数，每个参数都有一个导数，这一组导数组成梯度。

```text
参数：w1, w2, w3, b
梯度：[dw1, dw2, dw3, db]
```

梯度告诉模型：

```text
每个参数应该往哪个方向调整，loss 才会下降
```

## 13. 损失函数

损失函数用于衡量模型预测结果有多差。

例如真实房价是 150，模型预测是 130：

```text
误差 = 130 - 150 = -20
```

平方误差：

```text
(-20)^2 = 400
```

多个样本时，常用均方误差 MSE：

```text
MSE = 所有平方误差的平均值
```

Python 示例：

```python
import numpy as np

y_true = np.array([120, 150, 180])
y_pred = np.array([110, 160, 200])

mse = np.mean((y_pred - y_true) ** 2)
print(mse)
```

损失函数越小，说明模型预测越接近真实值。

训练模型的核心目标：

```text
找到一组参数，让损失函数尽可能小
```

## 14. 梯度下降

### 14.1 核心思想

梯度下降是一种寻找较小损失值的方法。

直觉：

```text
你站在山坡上
想走到山谷最低点
每一步都朝最陡的下降方向走一点
```

在模型训练中：

```text
山坡高度 = loss
当前位置 = 当前模型参数
下降方向 = 梯度的反方向
步长 = 学习率 learning rate
```

参数更新公式：

```text
new_parameter = old_parameter - learning_rate * gradient
```

### 14.2 学习率

学习率控制每次更新走多大一步。

```text
学习率太小：训练很慢
学习率太大：可能越过最低点，甚至发散
```

示意：

```text
太小：
当前位置 -> . -> . -> . -> 最低点

合适：
当前位置 -> -> -> 最低点

太大：
当前位置 -> 越过最低点 -> 来回震荡
```

## 15. 从零实现线性回归训练

现在用一个极简例子，把前面的概念串起来。

我们希望模型学会：

```text
y = 2x + 1
```

但模型一开始不知道 2 和 1，它只有随机参数：

```text
y_pred = w*x + b
```

目标是通过训练，让：

```text
w 接近 2
b 接近 1
```

### 15.1 完整代码

创建文件：

```text
src/linear_regression_from_scratch.py
```

写入：

```python
import numpy as np


def main() -> None:
    # 训练数据：真实规律是 y = 2x + 1
    X = np.array([1, 2, 3, 4, 5], dtype=float)
    y = np.array([3, 5, 7, 9, 11], dtype=float)

    # 初始化参数
    w = 0.0
    b = 0.0

    learning_rate = 0.01
    epochs = 1000
    n = len(X)

    for epoch in range(epochs):
        # 1. 前向计算：根据当前参数做预测
        y_pred = w * X + b

        # 2. 计算损失：均方误差
        loss = np.mean((y_pred - y) ** 2)

        # 3. 计算梯度
        dw = (2 / n) * np.sum((y_pred - y) * X)
        db = (2 / n) * np.sum(y_pred - y)

        # 4. 更新参数
        w = w - learning_rate * dw
        b = b - learning_rate * db

        if epoch % 100 == 0:
            print(f"epoch={epoch}, loss={loss:.6f}, w={w:.4f}, b={b:.4f}")

    print("Final model:")
    print(f"y = {w:.4f} * x + {b:.4f}")


if __name__ == "__main__":
    main()
```

运行：

```powershell
python src/linear_regression_from_scratch.py
```

你会看到 loss 逐步变小，w 接近 2，b 接近 1。

### 15.2 训练循环拆解

机器学习训练通常都有类似结构：

```text
for epoch in range(epochs):
    预测
    计算损失
    计算梯度
    更新参数
```

这也是后续 PyTorch 训练代码的核心结构：

```python
for epoch in range(epochs):
    y_pred = model(X)
    loss = loss_fn(y_pred, y)
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
```

区别是：

- NumPy 示例里，我们手写梯度。
- PyTorch 中，框架自动计算梯度。

但思想完全一致。

## 16. Embedding 相似度小实验

这里先不用真实大模型生成 Embedding，手动构造几个向量模拟语义。

创建文件：

```text
src/embedding_similarity_demo.py
```

写入：

```python
import numpy as np


def cosine_similarity(a: np.ndarray, b: np.ndarray) -> float:
    return float(np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b)))


def main() -> None:
    query = np.array([0.9, 0.8, 0.1])

    documents = {
        "Java Spring Boot 开发": np.array([0.88, 0.82, 0.12]),
        "机器学习模型训练": np.array([0.75, 0.70, 0.25]),
        "周末旅游攻略": np.array([0.10, 0.20, 0.95]),
    }

    scores = []
    for title, vector in documents.items():
        score = cosine_similarity(query, vector)
        scores.append((title, score))

    scores.sort(key=lambda item: item[1], reverse=True)

    for title, score in scores:
        print(f"{title}: {score:.4f}")


if __name__ == "__main__":
    main()
```

运行后你会看到和 query 方向最接近的文档排在前面。

这就是后续 RAG 检索的数学基础。
