# Java 开发者的 AI 基础理论与大模型应用开发实战

本目录用于沉淀一套面向 Java 开发者的 AI 系统课程。课程目标是让有 Java 开发基础、AI 基础较弱的学习者，从 AI 基础概念开始，逐步掌握机器学习、深度学习、大模型应用开发，以及 Java/Spring Boot 场景下的 AI 工程化落地。

## 目录说明

- `COURSE_TASKS.md`：任务拆解、课程进度、完成状态。
- `COURSE_OUTLINE.md`：完整课程大纲与学习流程。
- `COURSE_WRITING_STANDARD.md`：后续课程生成与既有课程回改规范。
- `SOURCES.md`：已使用的信息源、官方文档和学习资料。
- `glossary.md`：课程专有名词解释。
- `chapters/`：分章节教材正文。
- `projects/`：项目实战说明、代码骨架或练习要求。
- `exams/`：阶段测试题、综合考试和验收标准。

## 学习方式

建议按章节顺序学习。每完成一章后，先完成对应练习与自测题，再进入下一章。

当前已开始生成教材，进度以 `COURSE_TASKS.md` 为准。
