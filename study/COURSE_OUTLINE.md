# 课程大纲与学习流程

课程主题：Java 开发者的 AI 基础理论与大模型应用开发实战

适用对象：有 Java 开发基础，但 AI 基础较弱的学习者。

课程目标：

1. 理解 AI、机器学习、深度学习、大模型、RAG、Agent 的核心关系。
2. 能用 Python 阅读和修改基础 AI 训练代码。
3. 理解机器学习和深度学习的基本训练流程。
4. 能使用大模型 API、Embedding、向量数据库构建 AI 应用。
5. 能使用 Spring Boot、Spring AI、LangChain4j 将 AI 能力接入 Java 系统。
6. 能完成企业知识库问答系统和智能客服系统两个综合项目。

## 阶段 0：入门定位

推荐时间：1 周

学习目标：

- 搞清 AI / ML / DL / LLM 的关系。
- 理解训练、推理、微调、RAG 的区别。
- 确定 Java 开发者学习 AI 的主线。

课程：

- 第 0 课：AI 全局认知与学习定位

阶段产出：

- 能用自己的话解释 AI 应用开发和模型训练的区别。
- 能画出一个 RAG 应用的基本架构。

## 阶段 1：Python + 数学底座

推荐时间：3-4 周

学习目标：

- 掌握 AI 示例代码中常见 Python 写法。
- 理解向量、矩阵、概率、导数在模型训练中的作用。
- 能使用 NumPy、Pandas、Matplotlib、Jupyter Notebook 完成基础数据实验。

课程：

- 第 1 课：Python 与 AI 开发环境
- 第 2 课：AI 数学基础：向量、矩阵、概率与导数

阶段项目：

- 使用 Python 分析一个 CSV 数据集，并输出统计结果和可视化图表。

## 阶段 2：机器学习基础

推荐时间：4-6 周

学习目标：

- 理解传统机器学习从数据到模型的完整流程。
- 掌握常见监督学习、无监督学习和模型评估方法。

课程：

- 第 3 课：机器学习完整流程
- 第 4 课：监督学习：回归与分类
- 第 5 课：无监督学习、特征工程与模型评估

阶段项目：

- 房价预测或用户流失预测。

## 阶段 3：深度学习基础

推荐时间：4-6 周

学习目标：

- 理解神经网络如何训练。
- 掌握 PyTorch 基础代码结构。
- 理解 Transformer 为什么成为大模型基础。

课程：

- 第 6 课：深度学习与 PyTorch 基础
- 第 7 课：CNN、RNN 与 Transformer 入门

阶段项目：

- 使用 PyTorch 完成一个图片分类器。

## 阶段 4：大模型应用开发

推荐时间：6-8 周

学习目标：

- 掌握 Prompt、Embedding、RAG、Tool Calling、Agent 等大模型应用核心技术。
- 能从零实现语义搜索和知识库问答。

课程：

- 第 8 课：大模型基础、Token 与 Prompt Engineering
- 第 9 课：Embedding、语义搜索与向量数据库
- 第 10 课：RAG 知识库问答系统
- 第 11 课：Function Calling、Tool Calling 与 Agent

阶段项目：

- Markdown / PDF / Word 文档知识库问答系统。

## 阶段 5：Java AI 工程化

推荐时间：4-8 周

学习目标：

- 使用 Spring Boot、Spring AI、LangChain4j 构建 Java AI 应用。
- 掌握流式输出、权限控制、日志审计、成本控制、模型评估等工程能力。

课程：

- 第 12 课：Spring Boot + Spring AI
- 第 13 课：LangChain4j 应用开发
- 第 14 课：Java AI 工程化：流式输出、权限、日志、成本与评估

阶段项目：

- Spring Boot + Spring AI 智能客服系统。

## 阶段 6：综合项目与进阶

推荐时间：长期

学习目标：

- 完成可运行、可演示、可扩展的 AI 应用。
- 建立后续深入学习路线。

内容：

- 综合项目 1：企业知识库问答系统
- 综合项目 2：智能客服系统
- 阶段考试、自测题与最终验收
- 进阶方向：微调、LoRA、Agent、多模态、MLOps、模型部署、推理优化
